In this folder there are documents used in implemnting portions of the GSM PHY layer.
These documents include:
	- From the actual standard named after what they refer to
		- Channel Coding: How the GSM standard encodes data
		- General: The overview of the standard
		- Modulation: How to modulate the data before transmitting
		- Multiplexing: Includes information on what kind of data is contained within bursts
	
	- GMSK_Explained: Some background on the GMSK modulation format
	- SCH info: Contains info on burst data
	- slides_08_6_4: Information on systematic encoding 

Slides from: http://www.comlab.hut.fi/studies/3410/slides_08_6_4.pdf
